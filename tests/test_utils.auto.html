Test Utilities¶

test_utils.bsh¶

Utilities to help in test verifications

check_a¶
Arguments

  • $1 - Array name

  • [$2…] - Array values

Output

Return Value:

  • 0 - Equal

  • 1 - A value is not the same

  • 2 - The array had less values than specified

Check array values for unit tests

See also

elements.bsh cmp_elements_a

check_ra¶

Regex version of check_a

contiguous_a¶
Arguments

$1 - Array name to test

Output

Return Value:

  • 0 - Array is contiguous

  • 1 - Array is not contiguous

Check array values are contiguous for unit tests

Note

Uses eval
